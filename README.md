Api, связывает Roblox и Notion. Принимает запросы с информацией о донатах на аккаунты. Обновляет информацию о донатах в таблице Notion.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
